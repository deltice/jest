お困りですか？

JestはFacebookのJavaScriptツールのチームによりフルタイムで開発されています。チームメンバーは大抵在席しており質問に回答できます。

ドキュメントを見る

探していることを詳細なマニュアルとガイドから見つけてください。

  • Jestを始める手順を学ぶ。
  • Jestのトラブルシューティング。
  • Jestの設定について学ぶ。
  • API リファレンスの全部をみる。

コミュニティに参加する

同じJestのユーザに質問し、回答を見つけましょう。

  • Discordコミュニティの Reactiflux内の#jestチャンネルに参加してください。
  • コミュニティメンバーの多くがStack Overflowを利用しています。 jestjsタグがついたexisting questionsから探すか、 質問しましょう!

最新情報を確認する

Jestの最新情報はこちらから。

  • ツイッターで Jestをフォローする。
  • Jest blogを購読する。
  • changelogを確認する。
